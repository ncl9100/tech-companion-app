# Tech Companion App (ElderlyAI)

ElderlyAI is a mobile app designed to help elderly users interact with their smartphone more confidently. It features a friendly AI assistant that provides step-by-step guidance on using phone features like calling, texting, camera usage, and more — all with a focus on accessibility, simplicity, and voice interaction.

## 🧠 Key Features

- 🎙️ **Voice Assistant**: Ask questions via voice and get clear, concise help on phone tasks.
- 📸 **Step-by-step Guides**: Learn how to use your phone’s camera, messages, and settings.
- 📱 **Tech Support Mode**: Offers help for common tech issues with an easy interface.
- 🧓 **Accessibility First**: Large fonts, high-contrast visuals, and voice-first design.
- 💬 **Offline Tips**: Helpful guides that work without internet access.

## 📲 Built With

- **React Native** (via Expo)
- **Zustand** for state management
- **Flask** (Python) for backend APIs
- **OpenAI API** (or similar LLM) for natural language understanding

## 🛠️ Getting Started

### Prerequisites

- Node.js
- Yarn or npm
- Expo CLI
- Python 3.x (for Flask backend)

### Installation

1. Clone this repo:
  
   git clone https://github.com/your-username/tech-companion-app.git
   cd tech-companion-app
   
2. Install frontend dependencies:

yarn install
# or
npm install

3. Start the Expo development server:

npx expo start

Note: Update .env files with NewsAPI key and OpenAPI key
